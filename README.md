# Digital Footprint Analyzer 🔍

A cybersecurity web application that analyzes digital footprints and detects potential data breaches.

## 🚀 Features
- Email, IP, URL scanning
- API + Local scan modes
- Risk scoring system
- Live threat intelligence feed
- Cybersecurity dashboard

## 🛠 Tech Stack
- Java (Servlets, JSP)
- MySQL
- JDBC
- HTML, CSS, JS

## 📊 Modules
- Scanner
- Result Analyzer
- Dashboard (Command Center)

## 🎯 Purpose
Built for cybersecurity awareness and academic demonstration.
